Proyecto 2: Análisis y Predicción de Capacidad de Placas Atornilladas

En este proyecto, realicé un análisis de datos de pruebas experimentales de placas atornilladas. El objetivo principal fue explorar las características de los datos y desarrollar un modelo predictivo para la carga máxima (Fmax).

  • Estadísticos Descriptivos: Calculé las principales métricas para comprender la distribución de las variables.
  • Visualizaciones Clave: Generé histogramas para la distribución de Fmax, diagramas de dispersión para analizar la relación entre Fmax y la relación borde/diámetro (e1/do), y un mapa de calor para visualizar las correlaciones entre todas las variables.
  • Modelo Predictivo: Implementé un modelo de Regresión Lineal para predecir la carga máxima (Fmax) basándome en diversas características geométricas y de material de las placas, evaluando su rendimiento con métricas como RMSE y R².

Fuente de los datos: Adaptado de: Pascual, C., & Galán, S. (2021). Experimental data from bolted connections with different end distances. Zenodo. Enlace al Dataset en Zenodo

Proyecto 3: Análisis de Dinámica Estructural (Vibración Libre SDOF)

Este proyecto de Google Colab implementa el análisis completo de un sistema de un grado de libertad (SDOF) sometido a vibración libre no amortiguada. Se calcula la ecuación de movimiento utilizando el Principio de D'Alembert y se visualiza la respuesta del sistema en términos de desplazamiento, velocidad y aceleración.

  • Cálculos Detallados: Determinación de parámetros del sistema (masa, rigidez), cálculo de frecuencia natural y constantes de integración.
  • Visualización Completa: Graficación del desplazamiento, velocidad y aceleración a lo largo del tiempo, demostrando la respuesta dinámica del sistema.
  • Fundamentos Teóricos: Aplicación del Principio de D'Alembert para la formulación de la ecuación diferencial del movimiento.

Proyecto 4: Conversión Celsius a Fahrenheit con Red Neuronal Lineal

En este proyecto fundamental de Machine Learning, desarrollé una red neuronal simple utilizando TensorFlow y Keras para aprender la relación entre temperaturas en grados Celsius y Fahrenheit.

  • Arquitectura del Modelo: Se implementó una red neuronal con una única capa densa (fully connected) y función de activación lineal, demostrando cómo incluso modelos simples pueden aprender relaciones matemáticas directas.
  • Entrenamiento: El modelo fue entrenado con pares de datos Celsius-Fahrenheit, utilizando el optimizador Adam y la pérdida de error cuadrático medio (MSE) para ajustar los pesos y sesgos.
  • Generalización y Predicción: Una vez entrenado, el modelo es capaz de predecir la temperatura en Fahrenheit para nuevos valores en Celsius, ilustrando la capacidad de generalización de la red.
